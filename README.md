# NwJS - Test Mundijuegos with Flash

# Instalation
```sh
$ cd nwjs-test
$ npm install -g nw
$ npm install -g nw-builder
$ npm install nw-builder --save-dev
```

# Run
```sh
$ npm start
```


# Build as a native application

```sh
$ npm install
```

# # On Windows
```sh
$ cd build\NwMaluco\win32\
$ \.NwMaluco.exe
```

# # On MacOS
```sh
$ cd build/NwMaluco/osx64
$ open NwMaluco.app
```
